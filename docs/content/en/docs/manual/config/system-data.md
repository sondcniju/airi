# System & Data

This area allows you to manage the application's global configuration and the data it stores.

## Memory

**Memory** provides an overview of the short-term and long-term memory systems for AIRI.

- **Conversation Context**: View and inspect the current conversation's content and summary.
- **Stored Memories**: Search through the long-term memory storage used for personal interaction.

---

## Data Management

The **Data** section is where you manage the AIRI lifecycle and exports.

- **Export/Import Chats**: Backup or restore your conversation history.
- **Delete All Sessions**: Clear all conversation sessions from the local database.
- **Remove All Models**: Delete all uploaded visual models to free up space.

### Danger Zone

The **Danger Zone** contains irreversible actions used to reset the system.

- **Reset All Modules**: Reset all module-specific settings and credentials.
- **Delete All Data**: Completely wipe all local settings, providers, and models.
- **Reset Desktop Settings**: Clear the application's visual and window state.

---

## System Settings

These are the global settings that govern the overall application behavior.

- **Theme & Appearance**: Switch between light and dark modes and select color schemes.
- **Language**: Change the user interface language and display settings.
- **Shortcuts**: Customize global keyboard shortcuts for the application.
- **Developer Options**: Advanced settings for debugging and testing.
- **Network & WebSocket**: Configure the internal communication bridge between the app and its background services.
