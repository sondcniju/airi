# Settings Overview

AIRI is designed to be highly configurable, allowing you to tailor the AI's personality, intelligence, and visual expression to your liking.

## Configuration Categories

The settings are organized into four main categories:

| Category | Description |
| :--- | :--- |
| **Character & Scene** | Configure your AI's identity, behavior model, and visual environment. |
| **Intelligence** | Set up the "brain" (LLM), speech synthesis, hearing, and specialized modules. |
| **Information** | Inspect the memory systems (short-term and long-term) and journal. |
| **System** | Manage global application behavior, shortcuts, and data lifecycle. |
