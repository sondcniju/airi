# AIRI v0.9.1-stable (April 20, 2026)

This release focuses on memory infrastructure stabilization, enhanced VRM interactions, and significant UI improvements.

### 🧠 Memory Hub & Consoles
- **Neurological Memory Hub**: Composed of a high-fidelity Mixed Rail interface and tiered retrieval terminal.
- **Dream State Lifecycle**: Implemented full dream state gating and background processing driven by raw chat logs.
- **Subject Lain Integration**: Finalized lifetime provisioning for the Memory Lab pipeline.

### 👗 VRM Interaction Engine
- **Tactile Discovery**: Implemented zero-latency tactile interaction with precomputed mesh discovery.
- **Wardrobe Discovery**: Real-time reactive discovery for outfits and blendshapes, anchored in the Controls Island UI.
- **Physics Enhancements**: Backported 'Pure Home' physics stability and surgical mesh export logic.

### 🍱 UI & Chat
- **Chatbox 2.0**: Modern high-density chat interface with optimized bubble layout and action menus.
- **Director Decisions**: Visual toast notifications for director-level autonomous decisions.
- **Consolidation**: Unified coordinate spaces for visual effects and consolidated UI components across windows.

### 🔧 Stability & Fixes
- **Locale Persistence**: Resolved a localized bug where Traditional Chinese preferences were reset to Simplified Chinese on reboot.
- **TTS Pipeline**: Implemented missing lifecycle hooks in `ChatOrchestrator` to stabilize the TTS voice stack.
- **Nix Support**: Updated pnpm dependency hashes for reproducible builds on NixOS.
